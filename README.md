[参考](https://jingyan.baidu.com/article/359911f5a4fe4b57fe03060d.html)

- `git status` 查看当前状态
- `git add .` 或者 `git add xxx`
- `git commit -m 'XXX'` 推送修改到本地git库中
- `git pull <远程主机名> <远程分支名>`
- `git push <远程主机名> <远程分支名>`
